搭建hexo博客

发表于 2020-10-29 分类于 搭建博客

hexo搭建博客

前提

已经搭建好github主页：username.github.io

环境配置

  • hexo文档：https://hexo.io/zh-cn/docs/

  • 所需软件：

    • Git
    • hexo
      • node.js

  • 安装Git

  • 安装node.js 官网： http://nodejs.cn/ 下载：http://nodejs.cn/download/

    • 将nodejs添加到环境变量
    • CMD里面输入node+回车不报错

  • 安装hexo

    • 保证nodejs正确安装后在cmd里输入
      npm install -g hexo-cli

    • 安装完成后检查是否安装成功
      hexo -v 返回版本号则正确安装

    • 初始化hexo 在磁盘中找一个位置
      hexo init myblog 其中myblog可以随便写
      cd myblog 进入到该文件夹
      npm install 完成后初始化成功

      ls命令查看目录 此目录之后称为根目录

    • node_modules: 依赖包

    • public：存放生成的页面

    • scaffolds：生成文章的一些模板

    • source：用来存放你的文章

    • themes：主题

      • ** _config.yml: 博客的配置文件**

        本机试用官方博客

  • 在上面的根目录下打开cmd
    hexo server 默认情况下在4000端口

  • 浏览器里输入localhost:4000查看博客

  • ctrl+c关闭服务

将本机博客部署到github

前提已经有了github page

  • 在根目录下找到_config.yml，该文档为hexo的配置文件，打开

  • 在文档最后，更改代码，示例

    deploy:
type: git
repo: git@github.com:username/username.github.io.git
branch: main```

  username是自己的账户名

  • 在cmd中加载部署模块
    npm install hexo-deployer-git --save

  • 上传到github
    hexo deploy 可缩写 hexo d
